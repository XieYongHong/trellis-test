# 项目页面设备分组子页

## Goal

在 `modules/page-ui` 中实现设备分组子页，让用户按区域或业务分组查看设备集合的在线情况和健康度。

## Requirements

* 页面挂载在 `page-ui` 的设备分组子路由。
* 以分组卡片或摘要列表展示设备分组。
* 每个分组展示名称、范围、在线数量、总量和健康度。
* 健康度用 chip/progress 等状态化表达。
* 提供查看分组动作，但不进入完整分组详情。

## Acceptance Criteria

* [ ] 设备分组子路由可访问。
* [ ] 至少展示 4 个分组。
* [ ] 每个分组能看出在线比例和健康度。
* [ ] 分组状态不用单侧彩色线条表达。
* [ ] 移动端卡片单列展示且文本不溢出。

## File Scope

* `modules/page-ui/views/project/device-groups/index.vue`

## Out of Scope

* 分组树拖拽
* 分组编辑
* 设备移入移出分组
