# 项目页面视频资源子页

## Goal

在 `modules/page-ui` 中实现视频资源子页，让安保/运维用户按状态、区域和通道识别摄像头资源。

## Requirements

* 页面挂载在 `page-ui` 的视频资源子路由。
* 使用资源卡片台账展示摄像头/通道。
* 卡片展示名称、通道、存储周期、区域/责任组、在线状态和主动作。
* 视频资源状态优先展示，不做字段表格平铺。
* 主动作可为打开画面或定位。

## Acceptance Criteria

* [ ] 视频资源子路由可访问。
* [ ] 至少展示 3 个视频资源卡片。
* [ ] 离线资源有明确状态 chip。
* [ ] 卡片不承担完整编辑表单。
* [ ] 页面不依赖真实视频流即可渲染。

## File Scope

* `modules/page-ui/views/project/video-resources/index.vue`

## Out of Scope

* 真实视频播放
* 摄像头新增/编辑
* 通道详情页

## Definition of Done

* 只完成本任务范围内的实现或整理。
* 通过相关构建/类型检查，或明确记录未执行原因。
* 若发现范围需要扩大，先更新 PRD 或返回父任务拆解，不直接顺手实现。
