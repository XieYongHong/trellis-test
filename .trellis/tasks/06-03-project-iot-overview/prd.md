# 项目页面物联设备总览子页

## Goal

在 `modules/page-ui` 中实现设备总览子页，让设备管理员快速理解设备规模、在线状态和重点设备。

## Requirements

* 页面挂载在 `page-ui` 的设备总览子路由。
* 展示设备状态分布：在线、离线、预警、故障。
* 展示设备异常摘要，优先复用异常列表组件。
* 展示重点设备卡片，卡片突出状态、标签、归属和主动作。
* 不做完整设备 CRUD。

## Acceptance Criteria

* [ ] 设备总览子路由可访问。
* [ ] 设备状态分布有计数和占比。
* [ ] 重点设备至少展示 3 张卡片。
* [ ] 单卡核心字段不超过 5 个，标签不超过 3 个。
* [ ] 主动作就近放在卡片底部或行内。

## File Scope

* `modules/page-ui/views/project/iot-overview/index.vue`

## Out of Scope

* 设备新增/编辑/删除
* 设备详情页
* 真实消息趋势接口

## Definition of Done

* 只完成本任务范围内的实现或整理。
* 通过相关构建/类型检查，或明确记录未执行原因。
* 若发现范围需要扩大，先更新 PRD 或返回父任务拆解，不直接顺手实现。
