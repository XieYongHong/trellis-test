# 项目页面画面与回放子页

## Goal

在 `modules/page-ui` 中实现画面与回放子页，让用户查看重点实时画面和事件回放片段入口。

## Requirements

* 页面挂载在 `page-ui` 的画面与回放子路由。
* 展示 1/4/9 画面模式切换控件。
* 展示多画面区域，可用无真实流的画面状态占位。
* 展示事件片段列表，包含类型、标题、摄像头、时间和回放动作。
* 离线画面有明确状态表达。

## Acceptance Criteria

* [ ] 画面与回放子路由可访问。
* [ ] 多画面区域在桌面至少 2 列，移动端单列。
* [ ] 事件片段至少 3 条。
* [ ] 回放动作就近展示。
* [ ] 不使用假 URL 触发真实播放器报错。

## File Scope

* `modules/page-ui/views/project/screen-playback/index.vue`

## Out of Scope

* 真实播放器接流
* 历史回放接口
* 云台控制

* 只完成本任务范围内的实现或整理。
